# Playwright Enterprise Framework

Modular core (browser, API, DB, AI) plus integrations and templates. See `docs/getting-started/`.

## Quick start

```bash
npm install
npx playwright install
```

## Layout

- `core/` — runtime utilities
- `ai/` — optional AI-assisted workflows
- `config/` — env and Playwright config slices
- `templates/` — copy-paste starters
